### Project: Numeric Comparison (A vs B)

**Description:**  
This VI compares two numeric inputs (A and B) and displays their relationship as well as the difference between them.
Note: The VI currently runs with Run Continuously for demonstration purposes.

**Features:**

1. **Smaller Number**  
   - Displays whether A or B is smaller.

2. **Larger Number**  
   - Displays whether A or B is larger.

3. **Equality Check**  
   - Compares A and B.  
   - Returns **YES** if they are equal, otherwise **NO**.

4. **Absolute Difference**  
   - Calculates and displays |A - B|.

**Controls:**  
- Numeric Control A  
- Numeric Control B  

**Indicators:**  
- Smaller Number Indicator  
- Larger Number Indicator  
- Equality Check (YES/NO)  
- Difference Indicator (|A - B|)  

**Concepts:**  
- Numeric Comparison Functions (`<`, `>`, `=`)  
- Boolean Indicators (YES/NO)  
- Absolute Value Function (|A - B|)
