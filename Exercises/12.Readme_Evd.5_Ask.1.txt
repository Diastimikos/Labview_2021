### Project: Conditional Addition/Subtraction

**Description:**  
The VI allows the user to input two numbers (**A** and **B**) and performs a calculation based on the position of a **Switch**.  
- If the Switch is **True**, the program **adds** A and B.  
- If the Switch is **False**, the program **subtracts** B from A.  
The VI runs **inside a While Loop** and continuously updates the **Result** until the **Stop Button** is pressed.

**Controls:**  
- **Numeric Input A** – first number.  
- **Numeric Input B** – second number.  
- **Switch** – selects addition (True) or subtraction (False).  
- **Stop Button** – stops the While Loop and terminates the VI.

**Indicators:**  
- **Result** – displays the outcome of the selected operation (A + B or A - B).

**Concepts:**  
- While Loops
- Numeric Controls & Indicators  
- Case Structures  
- Boolean Controls (Switch)  
- Conditional Operations (Addition/Subtraction)
