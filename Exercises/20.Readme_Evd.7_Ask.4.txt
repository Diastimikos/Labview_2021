### Project: User Information Lookup

**Description:**  
The program demonstrates **searching and displaying user information** inside a While Loop:  
- The loop runs continuously until the **Stop Button** is pressed.  
- Three arrays store predefined values:  
  - **Name**  
  - **Last Name**  
  - **Birth Date**  
- The user inputs:  
  - **Element Category:** (name / lastname / birthdate / alldata)  
  - **Name:** the name of the user to search  
- Behavior:  
  - If the name exists in the arrays, the corresponding information is displayed according to the **Element Category**:  
    - `name` → displays Name  
    - `lastname` → displays Last Name  
    - `birthdate` → displays Birth Date  
    - `alldata` → displays all information together  
  - If the name does not exist, nothing is displayed.  
  - If both **Element Category** and **Name** are empty → displays `"Element Category and Element not defined"`  
  - If only **Element Category** is empty → displays `"No Representation Entry set"`  

**Controls:**  
- Element Category (string input)  
- Name (string input)  
- Stop Button  

**Indicators:**  
- Result – displays the requested information or messages  

**Concepts:**  
- Case Structures  
- Logical Gates
- Comparison Functions  
- Selectors 
- Search 1D Array 
- Index Array 
- Concatenate Strings
