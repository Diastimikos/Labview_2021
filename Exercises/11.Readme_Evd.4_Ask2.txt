### Project: Temperature Conversion (Celsius ↔ Fahrenheit ↔ Kelvin)

**Description:**  
The user inputs a temperature value (via arrows or keyboard, allowing integers and decimals) in Celsius.  
The program automatically converts and displays the value in **Fahrenheit**, **Celsius**, and **Kelvin** using three thermometers and three numeric indicators.  
The VI runs **inside a While Loop** and continuously updates the conversions until the **Stop Button** is pressed.

**Controls:**  
- Numeric Input (Celsius, via arrow buttons or keyboard)

**Indicators:**  
- Thermometer (Celsius)  
- Thermometer (Fahrenheit)  
- Thermometer (Kelvin)  
- Numeric Indicator (Celsius 2)  
- Numeric Indicator (Fahrenheit 2)  
- Numeric Indicator (Kelvin 2)  

**Concepts:**  
- While Loop 
- Numeric Controls & Indicators  
- Numeric Functions  
- Unit Conversion (°C ↔ °F ↔ K)
